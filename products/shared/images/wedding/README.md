# Wedding Photo & Icon Library — Ảnh mẫu + Icon cho thiệp cưới

Kho ảnh và icon dùng chung khi gen thiệp cưới mẫu.
Mỗi subfolder = 1 bộ ảnh theo phong cách/tông màu riêng.

## Cách dùng

Khi gen mẫu thiệp cưới (`/gen-wedding`):
1. Chọn bộ ảnh phù hợp phong cách/tông màu thiệp
2. Dùng relative path từ folder sản phẩm: `../../../shared/images/wedding/{bộ-ảnh}/{file}`
3. Chọn icon phù hợp: `../../../shared/images/wedding/icons/{file}`

## Thêm bộ ảnh / icon mới

### Thêm bộ ảnh couple:
1. Tạo subfolder: `{phong-cách}-{mô-tả}/`
2. Đặt 6-13 ảnh WebP (~30-200KB mỗi ảnh)
3. Đặt tên file: `couple_{số}.webp` hoặc tên mô tả (`bride.webp`, `story_1.webp`)
4. Cập nhật README này
5. Dùng relative path: `../../../shared/images/wedding/{bộ-ảnh}/{file}`

### Thêm icon:
1. Đặt file vào `icons/` với tên kebab-case
2. Format: WebP hoặc SVG
3. Cập nhật bảng icons trong README
4. Dùng relative path: `../../../shared/images/wedding/icons/{file}`

## Lưu ý

- Ảnh couple chỉ dùng làm MẪU — khi gen cho khách thật sẽ thay bằng ảnh khách
- Ưu tiên ảnh cặp đôi châu Á (Hàn/Trung/Việt) — phù hợp thị trường VN
- Icons dùng lại được cho nhiều mẫu — chọn theo phong cách thiệp
- Khi thêm sản phẩm mới: reference trực tiếp shared thay vì copy file
