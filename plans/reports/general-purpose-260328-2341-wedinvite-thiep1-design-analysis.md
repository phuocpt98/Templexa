# Design Analysis Report: wedinvite.online/thiep1

**URL:** http://www.wedinvite.online/thiep1
**Title:** Thiep moi cuoi Thanh Hai & Thu Nga 1
**Platform:** LadiPage (page builder)
**Total Page Height:** ~4027px (mobile 390x844)
**Date Analyzed:** 2026-03-28

---

## Overall Design Style

- **Mood:** Elegant, organic, natural, romantic Vietnamese wedding
- **Style:** Garden/botanical wedding with white-green color scheme
- **Texture:** White floral embossed pattern background throughout (like pressed linen/paper)
- **Feel:** Traditional Vietnamese wedding with modern layout, clean and airy

## Typography

### Font Stack
| Font | Type | Usage | Size Range |
|------|------|-------|------------|
| **MeaCulpa Regular** | Script/handwritten | Couple names on hero (main title) | 34px |
| **Dancing Script** | Script/cursive | Bride & groom individual names | 30px |
| **FleurDeLeah Regular** | Script/decorative | Guest name placeholder, decorative text | 25px |
| **LavishlyYours Regular** | Script/romantic | "Album Hinh Cuoi" title | 28px |
| **Playfair Display** | Serif/display | RSVP section title | 28px, bold |
| **Cormorant Upright** | Serif/elegant | Event details, section titles, addresses | 16-19px |
| **Tinos** | Serif/classic | Family information, labels | 15px |
| **Philosopher** | Sans-serif | Form labels, button text | 14-16px, bold |
| **EB Garamond** | Serif/classic | Date display | 20px |
| **Mulish** | Sans-serif | Radio button labels | 16px |
| **Open Sans** | Sans-serif | Footer, small links | 12-14px |
| **Lora Variable** | Serif | Body text (loaded but minor usage) | - |

### Typography Hierarchy
1. **Hero names:** Script font (MeaCulpa), 34px, regular weight, black
2. **Section titles:** Serif (Cormorant Upright/Playfair Display), 18-28px, bold, black or white
3. **Names:** Script (Dancing Script), 30px, regular
4. **Body/details:** Serif (Cormorant Upright), 16-18px, regular
5. **Labels:** Serif (Tinos) or sans (Philosopher), 14-16px, bold
6. **Small text:** Sans-serif (Open Sans), 12px

## Layout Patterns

1. **Single column, centered:** Dominant layout - everything vertically stacked and centered
2. **Two-column split:** Used for NHA GAI / NHA TRAI family info (symmetrical)
3. **Photo mosaic/collage:** Hero section - creative grid with varied photo sizes
4. **Photo gallery grid:** Album section - uniform grid

## Technical Notes

- Built on LadiPage platform (Vietnamese page builder)
- Fonts served from `w.ladicdn.com` CDN
- Images served from same CDN with responsive sizing (s768x, s650x, s400x prefixes)
- Base64-encoded font-family names in CSS (LadiPage obfuscation)
- HSTS enabled, but SSL cert has issues (common name mismatch)
- Total of ~40+ wedding photos used across collage, inline photos, and gallery
- Page is fully mobile-responsive (designed mobile-first at 390px width)
